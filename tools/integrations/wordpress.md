# WordPress

## What It Does
WordPress is a content management system for creating and managing websites, blogs, and digital content. It supports plugins, themes, and custom functionality.

## Key Capabilities for Marketing
- **Blog publishing** - Create and manage blog posts
- **Content management** - Organize pages, posts, and media
- **Plugins** - Extend functionality with marketing tools
- **SEO tools** - Optimize content for search engines
- **User management** - Control access and permissions
- **Analytics** - Track traffic and content performance

## How to Connect via MCP
1. Log into WordPress admin (wp-admin)
2. Go to Settings > Application Passwords
3. Generate an application password
4. Copy username and password
5. Note your WordPress site URL
6. Configure MCP with credentials

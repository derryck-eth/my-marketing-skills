# Stripe

## What It Does
Stripe processes payments and manages subscriptions, invoicing, and financial data. It's a payment platform for accepting and tracking revenue.

## Key Capabilities for Marketing
- **Payment processing** - Accept credit cards, wallets, and regional payment methods
- **Subscriptions** - Manage recurring billing and customer churn
- **Invoicing** - Create and send invoices automatically
- **Revenue analytics** - Track payment metrics and revenue data
- **Refunds** - Process refunds and disputes
- **Customer management** - Maintain customer payment information

## How to Connect via MCP
1. Log into Stripe Dashboard
2. Go to Settings > API Keys
3. Copy your publishable and secret key
4. Restrict key permissions as needed
5. Configure MCP with secret key
6. Enable webhook events for real-time updates

## Example Agent Queries
- "What's our total revenue for the month?"
- "Show me customers with failed subscription renewals"
- "What's our average order value and customer lifetime value?"
- "Create a new subscription for customer [ID]"
- "Generate a revenue report by product/plan"

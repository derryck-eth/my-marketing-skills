# Kit (formerly ConvertKit)

## What It Does
Kit is a creator email marketing platform providing email campaigns, automations, landing pages, and monetization tools for creators and solopreneurs.

## Key Capabilities for Marketing
- **Email campaigns** - Send branded emails to subscribers
- **Automations** - Build email sequences triggered by actions
- **Landing pages** - Create conversion-focused pages
- **Forms** - Embed signup forms on websites
- **Subscriber management** - Manage and segment audiences
- **Analytics** - Track opens, clicks, conversions, and revenue

## How to Connect via MCP
1. Log into Kit account
2. Go to Account > API Tokens
3. Create new API token
4. Copy token
5. Configure MCP with credentials
6. Grant access to broadcasts and automations

## Example Agent Queries
- "Send an email to subscribers who clicked a link"
- "Create an automation that triggers when someone joins"
- "What's the open rate on my recent email?"
- "Build a landing page for a product launch"
- "Show me revenue generated from a specific email"

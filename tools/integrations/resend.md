# Resend

## What It Does
Resend is an API-first email delivery platform for developers and teams. It simplifies transactional and marketing emails with high deliverability and templates.

## Key Capabilities for Marketing
- **Transactional email** - Send confirmations, receipts, and notifications
- **Email templates** - Use React-based templates for dynamic content
- **API delivery** - Programmatically send emails from applications
- **Webhooks** - Track bounces, opens, clicks, and delivery events
- **Analytics** - Monitor delivery rates, opens, and engagement
- **Domain authentication** - SPF, DKIM, and DMARC setup for deliverability

## How to Connect via MCP
1. Log into Resend dashboard
2. Navigate to API Keys section
3. Create a new API key
4. Copy the key
5. Configure MCP with API credentials
6. Verify sending domain

## Example Agent Queries
- "Send a password reset email to user [email]"
- "Create a welcome email template with dynamic content"
- "Track open and click rates for transactional emails"
- "Send a batch of order confirmation emails"
- "What emails bounced in the last week?"

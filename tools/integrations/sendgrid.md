# SendGrid

## What It Does
SendGrid is an email delivery platform offering transactional and marketing email capabilities with advanced analytics, templates, and sender reputation management.

## Key Capabilities for Marketing
- **Email API** - Send emails programmatically at scale
- **Dynamic templates** - Build emails with personalized content
- **Marketing campaigns** - Create and send targeted email campaigns
- **Automation** - Trigger emails based on user actions
- **Deliverability tools** - Manage sender reputation and avoid spam
- **Analytics** - Detailed metrics on opens, clicks, bounces, unsubscribes

## How to Connect via MCP
1. Log into SendGrid dashboard
2. Go to Settings > API Keys
3. Create new API key with required permissions
4. Copy the key
5. Configure MCP with API credentials
6. Verify sending domains

## Example Agent Queries
- "Send an invoice email with attachment to customer [email]"
- "Create a reactivation campaign for inactive users"
- "What's the bounce rate for our last broadcast?"
- "Build a welcome email series triggered on signup"
- "Send personalized product recommendations via email"

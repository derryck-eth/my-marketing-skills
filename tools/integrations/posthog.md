# PostHog

## What It Does
PostHog is an open-source product analytics platform combining event analytics, session recordings, heatmaps, and feature flag management in one tool.

## Key Capabilities for Marketing
- **Session recordings** - Watch how users interact with your product
- **Heatmaps and replays** - Visualize user behavior on web and mobile
- **Event tracking** - Capture and analyze all user interactions
- **Feature flags** - Control feature rollouts and run A/B tests
- **Cohort analysis** - Group users by behavior or properties
- **Data self-hosting** - Keep analytics data on your own infrastructure

## How to Connect via MCP
1. Access PostHog dashboard at app.posthog.com
2. Go to Settings > API Keys
3. Create a personal API key
4. Note your project ID and host URL
5. Configure MCP with API credentials
6. Grant appropriate data access permissions

## Example Agent Queries
- "Show session recordings of users who didn't convert"
- "What's the heatmap showing for our signup form?"
- "Create a cohort of users who activated feature X"
- "Which features have the highest adoption rate?"
- "Run an A/B test on the checkout flow"

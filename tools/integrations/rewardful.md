# Rewardful

## What It Does
Rewardful is an affiliate and referral tracking platform for SaaS companies. It manages commissions, affiliate payouts, and program performance.

## Key Capabilities for Marketing
- **Affiliate tracking** - Track affiliate referrals and conversions
- **Commission management** - Automate commission calculations
- **Affiliate recruitment** - Manage affiliate sign-ups and approvals
- **Payout management** - Distribute affiliate payouts automatically
- **Analytics** - Track affiliate performance and program ROI
- **Integrations** - Connect with payment processors and CRMs

## How to Connect via MCP
1. Log into Rewardful dashboard
2. Go to Settings > API Keys
3. Generate API key
4. Copy key and secret
5. Configure MCP with credentials
6. Set commission rates and payout rules

## Example Agent Queries
- "Show top 10 affiliates by conversion count"
- "What's the total commission owed for this month?"
- "Create a new affiliate with [email]"
- "Process payouts for affiliates above $100"
- "Get conversion report for [affiliate]"

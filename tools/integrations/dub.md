# Dub.co

## What It Does
Dub.co is a link management and analytics platform. It creates shortened links, QR codes, and tracks clicks, providing insights into link performance.

## Key Capabilities for Marketing
- **Link shortening** - Create branded short links
- **QR codes** - Generate QR codes for campaigns
- **Click analytics** - Track clicks, geographic data, and devices
- **Link customization** - Create memorable URLs with custom domains
- **Bulk operations** - Create and manage many links at once
- **Integrations** - Connect with marketing tools for tracking

## How to Connect via MCP
1. Log into Dub.co dashboard
2. Go to Settings > API Keys
3. Create or copy your API key
4. Configure MCP with API key
5. Set up tracking parameters for campaigns
6. Test connection

## Example Agent Queries
- "Create a shortened link for our campaign with UTM parameters"
- "Generate a QR code linking to [URL]"
- "Show me click analytics for link [short_url]"
- "Create 50 links for our email campaign"
- "What's the geographic breakdown of clicks on this link?"

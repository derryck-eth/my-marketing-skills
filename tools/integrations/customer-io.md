# Customer.io

## What It Does
Customer.io enables behavioral email marketing, push notifications, and SMS through customer journeys. It automates campaigns based on user actions and segment membership.

## Key Capabilities for Marketing
- **Behavioral triggers** - Send emails based on user actions in real-time
- **Journey automation** - Build multi-step workflows with conditions and delays
- **Push notifications** - Send native mobile notifications
- **SMS campaigns** - Include text messaging in customer journeys
- **Segmentation** - Target campaigns to specific user groups
- **Analytics** - Track opens, clicks, conversions for each campaign

## How to Connect via MCP
1. Log into Customer.io dashboard
2. Go to Settings > API Credentials
3. Create API key or use OAuth
4. Copy your Site ID and API key
5. Configure MCP with credentials
6. Test connection with sample campaign

## Example Agent Queries
- "Send an onboarding email sequence to new users"
- "Trigger an abandoned cart email when a user leaves without purchasing"
- "Create a segment of users inactive for 60 days and send re-engagement campaign"
- "Build a birthday email with personalized discount"
- "What's the open rate and click-through rate for last week's campaigns?"

# Mention Me

## What It Does
Mention Me is a referral marketing platform enabling brands to run referral programs that turn customers into advocates. It tracks referrals, rewards, and program performance.

## Key Capabilities for Marketing
- **Referral programs** - Create and manage referral incentive programs
- **Advocate tracking** - Identify and manage brand advocates
- **Reward management** - Distribute rewards to program participants
- **Analytics** - Track referral conversions, ROI, and program metrics
- **Integration** - Connect with CRM and marketing tools
- **Campaign management** - Create time-limited referral campaigns

## How to Connect via MCP
1. Log into Mention Me dashboard
2. Go to Settings > API
3. Generate API credentials
4. Copy API key and secret
5. Configure MCP with credentials
6. Select program to manage

## Example Agent Queries
- "Show me top referrers in our program"
- "What's the conversion rate of referred customers?"
- "Send reward to users who completed their referral"
- "Create a new referral campaign for [product]"
- "Get referral program analytics for last month"

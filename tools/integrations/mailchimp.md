# Mailchimp

## What It Does
Mailchimp is an email marketing platform enabling email campaigns, automation, audience segmentation, and analytics for businesses of all sizes.

## Key Capabilities for Marketing
- **Email campaigns** - Design and send branded email campaigns
- **Automation** - Set up workflows triggered by user actions or dates
- **List segmentation** - Create targeted audiences based on behaviors and data
- **A/B testing** - Test subject lines, content, and send times
- **Templates** - Pre-designed templates or custom HTML emails
- **Analytics** - Track opens, clicks, conversions, and revenue

## How to Connect via MCP
1. Log into Mailchimp account
2. Go to Account Settings > Extras > API Keys
3. Generate a new API key
4. Copy your server prefix (us1, us2, etc.)
5. Configure MCP with API key and server info
6. Grant access to contact lists

## Example Agent Queries
- "Send a promotional email to users who haven't purchased in 90 days"
- "Create an automated welcome series for new subscribers"
- "Segment the list by location and send geo-targeted campaigns"
- "What's the click-through rate on our last newsletter?"
- "A/B test subject lines and show me the winner"

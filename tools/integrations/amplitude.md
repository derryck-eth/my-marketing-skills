# Amplitude

## What It Does
Amplitude is a product analytics platform focused on user behavior tracking, journey mapping, cohort analysis, and retention metrics. It helps teams understand how users engage with products.

## Key Capabilities for Marketing
- **User journey analysis** - Visualize the complete path users take through your product
- **Retention cohorts** - Track how different user cohorts behave over time
- **Event tracking** - Monitor in-product events and user actions
- **Funnel analysis** - Identify where users drop off in conversion flows
- **Behavioral segmentation** - Create user groups based on actions and patterns
- **Predictive analytics** - Forecast churn and identify at-risk users

## How to Connect via MCP
1. Log into Amplitude dashboard
2. Navigate to Settings > Projects > API Keys
3. Copy your API key and Secret key
4. Configure MCP connector with credentials
5. Specify your Amplitude region (US or EU)
6. Test connection with sample event query

## Example Agent Queries
- "What's the user retention rate for Q4 cohort?"
- "Show me the funnel from signup to first purchase"
- "Which feature drives the highest engagement?"
- "Create a segment of users who churned in the last 30 days"
- "Compare user behaviors: free vs paid tiers"

# Google Ads

## What It Does
Google Ads manages search, display, shopping, and video advertising campaigns. It handles keyword bidding, audience targeting, conversion tracking, and performance measurement.

## Key Capabilities for Marketing
- **Search campaigns** - Bid on keywords and show ads in Google search results
- **Display campaigns** - Reach users across websites and apps
- **Shopping campaigns** - Advertise products with pricing and inventory
- **Audience management** - Target by interests, behaviors, and demographics
- **Bid automation** - Smart bidding strategies to maximize conversions
- **Conversion tracking** - Measure ROI and campaign effectiveness

## How to Connect via MCP
1. Log into Google Ads account
2. Go to Tools & Settings > Setup > Conversions
3. Create API credentials via Google Cloud
4. Enable Google Ads API in Cloud Console
5. Generate OAuth credentials or service account
6. Configure MCP with API credentials

## Example Agent Queries
- "Increase budget for top-performing keywords by 20%"
- "Create a shopping campaign for new products"
- "What's the average CPC and conversion rate by device?"
- "Build an audience targeting users who visited product pages"
- "Pause underperforming ads with low CTR"

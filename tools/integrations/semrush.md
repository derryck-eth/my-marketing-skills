# Semrush

## What It Does
Semrush is a comprehensive SEO tool for keyword research, competitive analysis, site audits, and rank tracking. It provides actionable insights for search visibility.

## Key Capabilities for Marketing
- **Keyword research** - Find search volume, difficulty, intent, and related terms
- **Competitive analysis** - Analyze competitor traffic sources and keywords
- **Site audit** - Identify and fix technical SEO issues
- **Rank tracking** - Monitor keyword positions and changes
- **Content analysis** - Evaluate content performance and recommendations
- **Backlink analysis** - Track referring domains and link quality

## How to Connect via MCP
1. Log into Semrush account
2. Navigate to Settings > API
3. Generate API key
4. Copy key and configure MCP
5. Select data access scope (keywords, competitors, etc.)
6. Test connection

## Example Agent Queries
- "Find 20 keywords with high search volume and low competition"
- "Analyze our competitors' organic traffic sources"
- "What's our current ranking for [keyword]?"
- "Create a list of technical issues from our site audit"
- "Show me the content performance on our competitors' top pages"

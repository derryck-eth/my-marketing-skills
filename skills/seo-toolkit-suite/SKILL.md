# SEO Toolkit Suite

## Purpose
SEO Toolkit Suite is an all-in-one SEO operations skill that combines the core functionality of a modern SEO tool stack into one execution system for agencies and in-house teams. It runs audit diagnostics, bulk URL analysis, side-by-side comparisons, schema generation, SERP snippet previews, and AI-era visibility workflows (AEO/GEO/AI search visibility), then converts findings into prioritized actions and implementation plans.

## When to Use
- You want one SEO workflow from diagnosis to implementation
- You need fast audits across many pages or client sites
- You need competitor and market context tied directly to SEO actions
- You want to prioritize fixes by business impact, not vanity metrics
- You need client-ready SEO reports with clear "what to fix next"
- You need AI-search optimization (AEO/GEO) in addition to classic SEO
- You need a repeatable SEO operating system for agency delivery

## Core Frameworks & Modules

### Module 1: SEO Scanner (Comprehensive On-Page Audit)
Run a broad on-page diagnostic covering high-impact technical and content factors:
- Title tags, meta descriptions, headings (H1-H6), and content depth
- Internal/external links, canonical tags, robots directives, sitemap presence
- Image alt coverage, social tags (Open Graph/Twitter), and schema presence
- Page speed, mobile readiness, HTTPS, and crawl/indexing blockers

Output:
- Severity-based findings (`Critical`, `Warning`, `Good`)
- Issue-level fix instructions and expected impact

### Module 2: Bulk Scan (Portfolio-Level Analysis)
Analyze multiple URLs together for agencies and large sites:
- Batch scan priority pages (service pages, landing pages, blog clusters)
- Normalize scoring across pages
- Identify repeated issues and systemic templates causing SEO debt

Output:
- URL-by-URL score table
- Cross-page issue frequency map
- Batch fix queue ordered by leverage

### Module 3: URL Compare (You vs Competitors)
Compare a target page against competitor pages:
- SERP positioning signals (content coverage, metadata strength, structure)
- Technical parity gaps (schema, indexability, speed, mobile)
- Intent alignment and conversion relevance gaps

Output:
- Comparison grid with winners/losers by category
- Positioning gap summary
- "Beat competitor" action stack

### Module 4: Schema and SERP Presentation
Ship structured data and snippet optimization:
- Generate/validate JSON-LD schema recommendations
- Improve snippet quality (title/meta) for CTR
- Validate social preview tags for share visibility

Output:
- Schema implementation plan by page type
- SERP snippet rewrite options
- Markup QA checklist

### Module 5: AI SEO Suite (AEO/GEO/AI Visibility)
Expand optimization beyond classic blue links:
- AEO: answer engine readiness (direct-answer formatting, FAQ structure)
- GEO: generative engine optimization (entity clarity and citation readiness)
- AI visibility scoring and competitor AI presence gap analysis

Output:
- AI visibility baseline + target score
- Answer-ready content changes by page
- AI citation and entity reinforcement plan

### Module 6: Reporting, Exports, and Execution Cadence
Convert analysis into execution:
- Build prioritized roadmap by impact and effort
- Create weekly sprint plan with owners and deadlines
- Track before/after metrics and ranking movement

Output:
- Client-ready report narrative
- 30-60-90 roadmap
- Weekly implementation board

### Module 7: Competitor + Market Research Engine
Produce deep competitor intelligence to drive strategy:
- Benchmark competitors across technical SEO, on-page quality, content depth, authority, local presence, and AI visibility
- Quantify share-of-voice and identify gap clusters
- Map opportunity themes by intent and revenue impact

Output:
- Competitor landscape matrix
- Market opportunity stack by theme
- Strategic implications and 90-day priorities

### Module 8: GSC Profile Audit + Snippet Lab
Operationalize performance loops directly from search data:
- Audit Search Console property setup, indexing hygiene, and segmentation
- Build query opportunity queue (high impression + low CTR, position 8-20, assisted-conversion terms)
- Generate and score title/meta variants for CTR lift

Output:
- GSC profile health checklist + governance cadence
- Query optimization queue
- Snippet variants with recommended winner

### Module 9: Content Analysis Benchmark Engine (Hobo-Web + Live Top-5 SERP)
Run URL-level content analysis against Google ranking-signal frameworks, then compare against current leaders:
- Parse target page structure (title, H1, intent cues, canonical, schema, section depth)
- Infer the primary query from H1/title/context if not provided
- Run live search for top 5 ranking pages for that query
- Analyze those top pages for structure, intent matching, trust/authority cues, and content effort patterns
- Benchmark findings against Hobo-Web reference frameworks (NavBoost, Mustang, T* topicality, Twiddlers, signal coherence)

Output:
- Top-5 SERP structural comparison table
- Benchmark scorecard by ranking-signal dimension
- Gap analysis and prioritized fixes
- 7/30/60-day implementation roadmap for the target URL

## Process (Step-by-Step)

### Phase 1: Context Intake (Required First Step)
Collect:
1. Business model, ICP, offers, and target geographies
2. Main SEO goals (traffic, leads, revenue pages, AI visibility)
3. Priority URLs and top competitors
4. Current stack/tools and known constraints
5. Baseline metrics (organic sessions, rankings, conversions)

### Phase 2: Diagnostic Run
1. Run SEO Scanner on core pages
2. Run Bulk Scan on priority URL set
3. Run URL Compare against 3-5 direct competitors
4. Assess schema and snippet quality
5. Assess AEO/GEO/AI visibility readiness
6. Run Content Analysis Benchmark on priority pages (live top-5 SERP + Hobo-Web framework checks)

### Phase 3: Prioritization and Planning
Score each initiative with:
- Impact (1-5)
- Confidence (1-5)
- Effort (1-5)

Priority formula:
`Score = (Impact * Confidence) / Effort`

Then classify:
- Quick wins (1-2 weeks)
- Core builds (3-6 weeks)
- Strategic initiatives (6-12+ weeks)

### Phase 4: Implementation
1. Apply critical technical fixes first
2. Upgrade highest-value pages (money pages + high-impression pages)
3. Deploy schema and snippet improvements
4. Ship AEO/GEO improvements on priority content
5. Run weekly QA checks and update progress board

### Phase 5: Measurement and Iteration
Track:
- Ranking movement on priority queries
- Organic CTR and impressions
- Conversion rate from organic landing pages
- AI visibility score and citation presence

Review cadence:
- Weekly implementation review
- Monthly business-impact review

## SEO Skill Mapping
Use this suite as an orchestrator and route deeper tasks into existing skills:
- Technical diagnosis -> `seo-audit`
- Page-level improvements -> `on-page-seo`
- Authority growth -> `link-building`, `topical-authority`
- Structured data -> `schema-markup`
- Scale workflows -> `programmatic-seo`
- Trust signals -> `eeat-optimization`
- Visual optimization -> `image-seo`
- Competitive SERP pages -> `competitor-alternatives`
